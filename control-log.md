## [2026-05-10] - Исправление критического бага: коллизии имён ассетов

### Проблема:
- Страница `v1/index.html` на GitHub Pages открывалась без стилей и картинок.
- **Причина**: функция генерации имён файлов использовала первые 16 символов base64 от URL — разные URL давали одинаковые имена (напр. `aHR0cHM6Ly9maW5h.css`), файлы перезаписывали друг друга.
- Итог: вместо 13 CSS-файлов сохранялось только 2 (последний перезаписывал все предыдущие).

### Исправление:
- **`scripts/scraper.js`**: генерация имён файлов переведена на `crypto.createHash('md5').update(absoluteUrl).digest('hex')` — каждый URL гарантированно получает уникальное имя.
- Добавлена проверка размера загруженной страницы (< 10KB → ошибка с понятным сообщением).
- Браузер переключён в режим `headless: false` для лучшего обхода антибот-защиты.
- Timeout загрузки увеличен с 60 до 90 секунд.
- Добавлена пауза 2 сек перед загрузкой страницы (имитация поведения человека).
