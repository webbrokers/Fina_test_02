# Fina Performance Testing Project

Проект для тестирования и оптимизации производительности страницы `finabank.ru`.

## Цели проекта
- Клонировать страницу `finabank.ru/collection/zajmy-s-plohoj-kreditnoj-istoriej/`.
- Опубликовать **v1** (оригинал) на GitHub Pages — замерить базовые показатели.
- Создать **v2** (оптимизированная) и улучшить Core Web Vitals.
- Сравнить результаты v1 vs v2 через Google PageSpeed Insights.

## Структура проекта

```
Fina_test_02/
├── scripts/
│   └── scraper.js       # Playwright-скрипт для клонирования страницы
├── v1/                  # Оригинальная версия (эталон)
│   ├── index.html
│   └── assets/          # Все статические ресурсы (CSS, JS, медиа)
├── v2/                  # Оптимизированная версия (в процессе)
├── control-log.md       # Журнал всех изменений
└── README.md
```

## Стек технологий
- **Playwright** + **Node.js** — только для скрейпинга (не финальный стек).
- **Статический HTML/CSS/JS** — финальный сайт для GitHub Pages.
